위젯티콘 개인정보처리방침

최종 업데이트: 2024년 12월

사주살롱(이하 "회사")은 「개인정보 보호법」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하게 처리하기 위해 다음과 같이 개인정보처리방침을 수립·공개합니다.

1. 수집하는 개인정보

수집 항목 수집 목적 보유 기간
위치 정보 (GPS 좌표) 현재 위치 기반 날씨 정보 제공 수집 즉시 처리 후 삭제 (저장하지 않음)
갤러리 사진 (사용자 선택 시) 커스텀 스티커 등록 사용자가 삭제할 때까지 기기 내 저장
중요: 위젯티콘은 사용자 데이터를 외부 서버에 저장하거나 제3자와 공유하지 않습니다. 모든 데이터는 사용자 기기에서만 처리됩니다.

2. 개인정보의 처리 목적

  • 위치 정보: 사용자의 현재 위치를 기반으로 실시간 날씨 정보를 표시합니다. 위치 정보는 날씨 API(기상청, OpenWeatherMap) 호출에만 사용되며, 앱 내에 저장되지 않습니다.
  • 갤러리 접근: 사용자가 직접 선택한 사진만 커스텀 스티커로 등록할 수 있습니다. 갤러리 전체를 스캔하거나 자동으로 업로드하지 않습니다.

3. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우에는 예외로 합니다:

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

4. 개인정보의 보유 및 파기

  • 위치 정보: 날씨 정보 조회 후 즉시 파기 (저장하지 않음)
  • 커스텀 스티커: 사용자가 앱 내에서 삭제하거나 앱을 삭제할 때 파기
  • 앱 설정 데이터: 앱 삭제 시 자동 파기

5. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 위치 권한 거부 또는 철회 (기기 설정 > 앱 > 위젯티콘 > 권한)
  • 저장된 커스텀 스티커 삭제 (앱 내 설정)
  • 앱 데이터 전체 삭제 (기기 설정 > 앱 > 위젯티콘 > 저장공간 > 데이터 삭제)

6. 광고 및 분석

위젯티콘은 Google AdMob을 통해 광고를 표시합니다. Google의 개인정보처리방침은 여기에서 확인할 수 있습니다.

광고 개인화를 원하지 않으시면 기기 설정에서 광고 ID를 재설정하거나 관심 기반 광고를 비활성화할 수 있습니다.

7. 앱 권한 안내

권한 사용 목적 필수 여부
다른 앱 위에 표시 플로팅 버튼 기능 선택 (플로팅 기능 사용 시)
위치 현재 위치 날씨 표시 선택 (날씨 기능 사용 시)
